# FlashPose Documentation

Welcome to the FlashPose documentation. FlashPose is a production-grade framework for human pose estimation, covering 2D/3D body pose, hand/face keypoints, action recognition, and gesture recognition.

## Key Features

- **Multiple Architectures**: ViTPose, HRNet, RTMPose
- **Multiple Tasks**: body, hand, face, whole-body, action, gesture
- **LoRA Fine-tuning**: Parameter-efficient adaptation
- **ONNX Export**: Production deployment
- **Real-time Inference**: RTMPose achieves 90+ FPS
- **Comprehensive Metrics**: PCK, AP, MPJPE, PA-MPJPE
